## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/superbahbi/twitter-clone-web.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run 
   ```sh
   expo start
   ```
